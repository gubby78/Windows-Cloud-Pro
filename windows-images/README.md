# Instructions to Upload Windows Ghost Spectre 11 Image

This document provides step-by-step instructions to upload the Windows Ghost Spectre 11 image to the repository.

## Prerequisites
- Ensure you have access to the repository `Windows-Cloud-Pro`.
- Windows Ghost Spectre 11 image file ready for upload.

## Steps to Upload
1. **Navigate to the `windows-images` Folder**:  
   Go to the `windows-images` directory in the `Windows-Cloud-Pro` repository.

2. **Click on the Upload Files Button**:  
   On the repository page, find and click on the `Upload files` button.

3. **Select the Image File**:  
   Drag and drop the Windows Ghost Spectre 11 image file or click `choose your files` to select the file from your local machine.

4. **Add a Commit Message**:  
   Provide a commit message relevant to the upload, such as `Upload Windows Ghost Spectre 11 image`.

5. **Commit Changes**:  
   Click on the `Commit changes` button to finalize the upload.

6. **Verify Upload**:  
   Check the `windows-images` directory to ensure that the Windows Ghost Spectre 11 image has been uploaded successfully.

## Additional Notes
- Make sure that the image is in the correct format and meets the repository’s requirements.
- If you encounter any issues, check the repository settings or contact the repository administrator for help.
